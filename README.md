# python_ddt_driven

###################################################

ddt框架为数据驱动主要针对同一个功能不同的数据入参测试，适用不同逻辑 \n
如在web下收入多个参数去搜索, 单个接口多次参数的输入 \n

主要包括：\n
ddt data, 写入代码中数据驱动 \n
ddt file，维护json或yaml数据驱动 \n
ddt txt, 维护txt读取text文件数据驱动 \n
还有读取xml，exel等数据驱动等 \n
